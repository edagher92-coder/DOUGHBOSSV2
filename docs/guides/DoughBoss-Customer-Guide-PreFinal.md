# How to order from DoughBoss

> **PRE-FINAL GUIDE:** The current concept demo is simulated. Online card payment is not confirmed as live yet. Follow the live ordering page where its labels differ from this draft.

## At a glance

**What you will do:** choose food, customise it, review your cart, place the order and keep your order number.

**Typical time:** about 3–5 minutes, depending on customisation.

**Have ready:** your contact details, collection/delivery information shown by the live site, and an optional voucher code.

`[PHOTO: Menu page showing categories and a product card]`

## 1. Choose your food

1. Open the DoughBoss ordering page.
2. Browse the menu and categories.
3. Select **Add to cart** for a simple item.
4. For a custom item, choose the available size, base or toppings before adding it.
5. Use the quantity controls if you need more than one.

**Pro tip:** Read the item description and dietary information. Contact the shop if you have an allergy; the ordering screen cannot guarantee there is no cross-contact.

`[ANIMATION NOTE: Show a customer opening an item, selecting options and adding it to the cart]`

## 2. Review “Your order”

1. Open **Your order** or the cart.
2. Check every item, option and quantity.
3. Remove anything you no longer want.
4. Confirm the subtotal, discount, fees and total shown.

`[DIAGRAM: Cart line → quantity → option summary → line price; subtotal → discount → total]`

## 3. Add a voucher, if you have one

1. Enter the code in **Voucher code**.
2. Select **Apply**.
3. Check that the discount appears before continuing.
4. Use **Remove** if you entered the wrong code.

Applying a voucher previews the discount. The voucher is consumed only when checkout completes successfully.

## 4. Enter checkout details

1. Choose the fulfilment option offered by the live site, such as **Pickup**.
2. Choose the shop when the site asks you to do so.
3. Enter your name, email and phone carefully.
4. Enter an address only if delivery is available and selected.
5. Add short order notes only when necessary.
6. Review the order one last time.

Pickup-window selection and final store timing rules: **TO BE CONFIRMED**.

## 5. Place the order

The final button may say **Place order**. If verified online payment is enabled later, it may instead show **Pay $…**.

> **PAYMENT INTEGRATION PENDING:** Do not assume the concept-demo payment button makes a real payment. Use only the verified options displayed on the live DoughBoss site.

1. Select the final order button once.
2. Wait for the confirmation screen.
3. If the connection drops, retry the same checkout instead of rebuilding the whole order. The current checkout foundation is designed to return the original completed order when the same attempt is retried.
4. Do not start another order until you have checked whether the first one succeeded.

`[SCREENSHOT: Real confirmation screen with order number and total; hide customer information]`

## 6. Save and track your order

1. Save the order number from the confirmation screen.
2. Open the order-tracking page.
3. Enter the order number and the same email used at checkout.
4. Follow the customer status shown.

Possible wording includes:

```text
Order received — waiting for the shop to accept
→ Accepted by the shop
→ Being prepared
→ Ready for pickup
→ Collected
```

When delivery is enabled, the final stages may instead show **Ready for delivery → On its way → Delivered**. Exact customer timing notifications remain **TO BE CONFIRMED**.

## Pro tips

- Keep your order number until the order is collected or delivered.
- Use the same email when tracking.
- Check the complete option summary before placing the order.
- Apply the voucher before the final checkout step.
- If you are unsure whether payment/order succeeded, contact the shop before trying a new payment.

## Quick fixes

| Problem | Try this |
|---|---|
| Ordering is closed | Try again during the hours displayed by the live shop. |
| Cart is empty | Add an item, then reopen **Your order**. |
| Voucher rejected | Check the code, expiry, minimum spend and whether it has already been used. |
| Tracking finds nothing | Check the order number and use the matching checkout email. |
| No confirmation after submitting | Retry the same checkout once; then contact the shop before creating a new order. |
| Delivery address is requested | Complete it, or switch to pickup if that option is available. |

Shop phone, support email, launch location and final service hours: **TO BE CONFIRMED**.
