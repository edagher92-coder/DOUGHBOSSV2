# DoughBoss 2.12.1 — install

**`doughboss-2.12.1.zip`** is the installable WordPress plugin.

## Update the plugin
1. WP Admin → **Plugins → Add New → Upload Plugin**
2. Choose `doughboss-2.12.1.zip` → **Install Now** → if it asks, **Replace current with uploaded** → **Activate**.

## Put the real menu live — ONE CLICK (no CLI)
**DoughBoss → Settings → Menu → "Import standard menu".**
Creates all 27 board items (Manoush, Pizza, Pies, Wraps, Desserts, Drinks) with prices,
categories and dietary flags. Safe to re-run.

## Staff tablets that never log out
DoughBoss → Settings → **Staff session (days)** → set e.g. `3650` → Save.

## Order/catering emails
DoughBoss → Settings → **Order notification email** → defaults to orders@doughboss.com.au.
